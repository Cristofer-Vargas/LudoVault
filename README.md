## Inspiração de Projeto
> "Como amante de jogos e entusiasta em tecnologia, me inspirei na plataforma ja existente usada pela [Twitch](https://www.twitch.tv/), a [IGDB](https://www.igdb.com/). IGDB é uma API gratuita que fornece uma biblioteca vasta de jogos disponíveis e ja disponibilizados pelo mercado, com informações de Desenvolvedoras, datas de lançamentos e atualizações, imagens, e muitos informações relacionada aos jogos, é a principal API do mercado que possui uma variedade tão grande e precisa desde jogos antigos até lançamentos recentes."
>
> "Inspirado nesse sistema, eu venho desenvolvendo esse projeto com intuíto de permitir que usuários possam regristras interesses e ter um local na internet onde possa salvar seus jogos, reunidos a partir de qualquer plataforma, ano e gênero, podendo ter listas nostálgicas com jogos antigos que memorizavam a infância, a playlists de jogos modernos que admiramos cada tecnologia e detalhe." ~ Cristofer Vargas
